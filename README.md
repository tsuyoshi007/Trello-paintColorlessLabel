# paintColorlessLabel

## Getting Started

### Prerequisites

### Inside .env file

* TRELLO_KEY=[string][1] (App_key)
* TRELLO_TOKEN=[string][1] (User_token)
* TRELLO_MBID=[string][1]  (MemberID)
* TRELLO_BOARDID=[string][1]


You also need :
  -*trello*
  -*dotenv*

### Installing

```
npm install
```

* Please specify the color you would like to paint the label in the functionCalling

## Running the tests

For Example:

```bash
node server.js
```
